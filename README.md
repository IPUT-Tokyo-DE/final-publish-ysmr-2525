# ここにゲームのタイトルを記述する
JackBetPuzlle　Royle
## ゲームの内容
3マッチパズル×ギャンブルを組み合わせた新感覚対戦パズルゲーム
プレイヤーは、ブロックをそろえて消去して、ゲージをためてスロットを回し、
抽選を当て、賞金を獲得。相手より多くお金を稼ぐ、知略とスリル、相対の感情を掻き立てる対戦
![画面イメージ](docs/images/game_image01.png)

## 操作説明
例
左スティック⇒移動
B⇒決定、ブロック移動
A⇒戻る
Y⇒ポインターの向き変更
Rトリガー⇒スロット開始
